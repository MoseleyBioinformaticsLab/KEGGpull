Pulling, Parsing, and Saving KEGG Entries

Functionality for pulling KEGG entries from the KEGG REST API, parsing them, and saving the entries as files.

class kegg_pull.pull.SinglePull(output: str, kegg_rest: Optional[KEGGrest] = None, entry_field: Optional[str] = None, multiprocessing_lock: Optional[Lock] = None)[source]

Class capable of performing a single request to the KEGG REST API for pulling up to a maximum number of entries.

Parameters

  • output (str) – Path to the location where entries are saved (treated like a zip file if ends in .zip, else a directory).

  • kegg_rest (KEGGrest) – Optional KEGGrest object used to make the requests to the KEGG REST API (a KEGGrest object with the default settings is created if one is not provided).

  • entry_field (str) – Optional KEGG entry field to pull.

  • multiprocessing_lock (Lock) – Blocks code writing to the same zip file among multiple processes. Used only if pulling to a zip file while multiprocessing.

class kegg_pull.rest.KEGGrest(n_tries: int = 3, time_out: int = 60, sleep_time: float = 5.0)[source]

Class containing methods for making requests to the KEGG REST API, including all the KEGG REST API operations.

Parameters

  • n_tries (int) – The number of times to try to make a request (can succeed the first time, or any of n_tries, or none of the tries).

  • time_out (int) – The number of seconds to wait for a request until marking it as timed out.

  • sleep_time (float) – The number of seconds to wait in between timed out requests or blacklisted requests.
